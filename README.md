📌 Project OverviewThis project involves the development of an end-to-end Machine Learning pipeline to predict 10 specific physical compost traits based on high-dimensional multivariate data. The dataset comprises 68 chemical, color, and spectral sensor features. The primary objective is to accurately map these complex chemical and spectral signatures to physical properties using robust feature selection and regression models.Author: Sarthak JhanwarRole: Research InternInstitution: IIT Kharagpur (Under Prof. S. Chakraborty)Timeline: Dec 2025 - Jan 2026📊 Dataset & PreprocessingInitial Data: 530 lab-tested compost samples.Features: 68 variables encompassing chemical properties, color metrics, and spectral sensor data.Targets: 10 distinct physical compost traits.Exploratory Data Analysis (EDA): Analyzed feature distributions, skewness, and generated correlation heatmaps to understand multi-collinearity.Data Cleaning: Applied a grouped Interquartile Range (IQR) method for outlier removal, refining the dataset down to 503 clean, high-quality samples for modeling.🛠️ Methodology & Feature EngineeringHandling 68 features required rigorous dimensionality reduction and feature selection to prevent overfitting and improve model interpretability. Two distinct approaches were benchmarked:Feature Selection: Extracted the top-5 predictors per target using Random Forest Feature Importance scores.Dimensionality Reduction: Applied Principal Component Analysis (PCA), capturing 95% of the variance using 7 principal components.🤖 ModelingThe selected features were fed into various machine learning algorithms to predict the continuous physical traits. All models were rigorously validated using 5-fold Cross-Validation to ensure generalizability.Algorithms Evaluated:Random Forest (RF)XGBoostK-Nearest Neighbors (KNN)📈 Key ResultsFeature Strategy Winner: Random Forest feature importance outperformed the PCA approach on 7 out of the 10 target variables.Accuracy / Variance Explained: $R^2$ scores across the targets ranged from 0.2 to 0.96, indicating excellent predictive capability for the best-performing traits.Error Rate: Achieved a best Normalized Root Mean Square Error (NRMSE) of 3.8%.
